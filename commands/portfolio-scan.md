# APEX Portfolio Activation Scan

**Request:** $ARGUMENTS

## Scan Protocol

### 1. Review Current Context

Check for:
- Outstanding tasks or commitments from previous sessions
- Upcoming deadlines in the next 2 weeks
- Any flagged items needing attention

### 2. Portfolio Status Check

For each portfolio, identify:

**BCCS District Leadership (60-70%)**
- Board meetings approaching
- MDE compliance deadlines
- PD sessions scheduled
- Systems change initiatives in progress

**Creative Ventures (15-20%)**
- Speaking engagements
- Writing projects
- Coaching commitments
- Thought leadership opportunities

**Business Development (10-15%)**
- Consulting prospects
- Revenue generating activities
- Brand building opportunities

**Non-Profit Planning (5-10%)**
- Community partnership needs
- Mission-driven projects

### 3. Priority Recommendations

Based on scan, recommend:
1. **Urgent this week:** Must be addressed in next 5 days
2. **Important this month:** Should be progressed
3. **Coming up:** Items to prepare for
4. **Opportunities spotted:** Potential synergies or strategic moves

### 4. Team Activation

Suggest which APEX teams should be activated:
- Grant Strategy - If deadlines or opportunities identified
- Strategic Communication - If board or public work needed
- Professional Learning - If PD sessions approaching
- Workflow Automation - If data or systems work needed
- Equity & Justice - If equity analysis required
- Quality Control - If high-stakes work pending
