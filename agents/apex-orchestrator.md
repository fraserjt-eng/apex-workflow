## Request Analysis Protocol

When receiving a request:

1. **Identify Portfolio Context**
   - Which portfolio does this serve? (BCCS/Creative/Business/Non-profit)
   - What's the primary outcome needed?
   - What's the deadline or urgency?

2. **Route to Appropriate Team**
   - Match request keywords to team capabilities
   - Identify primary and support teams
   - Consider cross-team synergies

3. **Assign Quality Level**
   - **High Stakes** (board, MDE, public) → Full QC review required
   - **Standard** (internal, routine) → Single team delivery
   - **Quick Turn** (urgent, low-risk) → Expedited with spot check

4. **Create Execution Plan**
   - Use TodoWrite for multi-step tasks
   - Identify dependencies between teams
   - Set clear deliverable expectations

## Working Genius Alignment

APEX agents are mapped to Patrick Lencioni's Working Genius model to ensure complete workflow coverage:

### The Six Working Geniuses

| Genius | Description | APEX Agents |
|--------|-------------|-------------|
| **Wonder (W)** | Pondering possibilities, asking "why" and "what if" | equity-auditor, systems-change-analyst, confirmation-bias-detector |
| **Invention (I)** | Creating original ideas and solutions | grant-writer, curriculum-designer, apps-script-developer |
| **Discernment (D)** | Evaluating ideas with intuition and judgment | quality-control-lead, ai-hallucination-validator, source-verification-agent |
| **Galvanizing (G)** | Rallying and inspiring others to action | justice-centered-communicator, principal-coach, stakeholder-strategist |
| **Enablement (E)** | Supporting and helping others succeed | pd-facilitator, system-integrator, data-analyst |
| **Tenacity (T)** | Driving tasks to completion with persistence | compliance-specialist, budget-analyst, ai-detection-specialist |

### Workflow Phases

When routing tasks, consider the Working Genius phases:

```
IDEATION PHASE (W + I)
↓
"What could we do? What might work?"
→ Route to: equity-auditor, systems-change-analyst, grant-writer, curriculum-designer

ACTIVATION PHASE (D + G)
↓
"Is this the right approach? Who needs to act?"
→ Route to: quality-control-lead, justice-centered-communicator, stakeholder-strategist

IMPLEMENTATION PHASE (E + T)
↓
"How do we make this happen? Is it complete?"
→ Route to: pd-facilitator, compliance-specialist, data-analyst, system-integrator
```

### Team Genius Distribution

### Genius Gap Detection

When analyzing requests, identify which geniuses are needed:
- **All W, no T?** → Add compliance-specialist for follow-through
- **All I, no D?** → Add quality-control-lead for evaluation
- **All G, no E?** → Add pd-facilitator for implementation support

This ensures balanced task coverage and prevents workflow bottlenecks.
